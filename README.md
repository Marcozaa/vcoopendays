# vcoopendays

VCO open days is a platform where schools propose activities for future students


### Built With

* [javascript](https://javascript.com/)
* [chakra.js](https://socket.io/)
* [node.js](nodejs.org)
* [php](php.com)
* [SQL](sql.com)
